# 🧺 Cleany Sistem Laundry

> ✨ Smart Laundry Management System built with JavaFX to streamline operations, enhance customer experience, and simplify transaction handling.

---

## 🚀 Overview

**Cleany Sistem Laundry** adalah aplikasi desktop berbasis Java yang dirancang untuk membantu operasional bisnis laundry menjadi lebih **efisien, terstruktur, dan modern**.

Aplikasi ini menyediakan sistem terintegrasi mulai dari **manajemen pelanggan, pesanan, hingga pembayaran digital (QRIS)** dalam satu platform yang user-friendly.

---

## 💡 Why This Project?

Di era digital, banyak usaha laundry masih menggunakan sistem manual yang:

- Rentan kesalahan pencatatan  
- Sulit melakukan tracking pesanan  
- Tidak efisien dalam transaksi  

👉 **Cleany hadir sebagai solusi digital** untuk meningkatkan produktivitas dan profesionalisme bisnis laundry.

---

## ✨ Key Features

### 🔐 Authentication System
- Login & Register user  
- Role-based access (Admin & Seller)  

### 👥 Customer Management
- Tambah & kelola data pelanggan  

### 📦 Order Management
- Input & tracking pesanan laundry  
- Sistem terstruktur untuk workflow laundry  

### 💳 Digital Payment Integration
- Popup pembayaran (QRIS)  
- Simulasi sistem pembayaran modern  

### 🧺 Service Management
- Pengelolaan jenis layanan laundry  

### 🖥️ Interactive UI
- Dibangun dengan JavaFX  
- Tampilan clean & mudah digunakan

## 🧠 System Architecture

Project ini menggunakan pendekatan **MVC (Model-View-Controller)**:

- **Model** → Pelanggan, Pesanan, Layanan, Admin  
- **View** → File `.fxml` (UI)  
- **Controller** → LoginController, PrimaryController, dll

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/username/cleany-sistem-laundry.git
cd cleany-sistem-laundry
```

### 2. Run dengan Maven
```bash
mvn clean install
mvn javafx:run
```

Atau jalankan langsung:
```bash
App.java
```
